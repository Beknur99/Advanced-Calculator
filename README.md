# Advanced Console Calculator

A simple yet functional **console-based calculator** written in Python.  
Supports basic operations and a few advanced ones.

## Features
- Addition (+)
- Subtraction (-)
- Multiplication (×)
- Division (÷) with zero-division handling
- Power (x^y)
- Square Root (√x)

## Version
